Niscord

Node + Discord = Niscord

About

Niscord is a powerful, easy-to-use JavaScript library for Discord API.

  • Powerful
  • Object-oriented
  • Easy-to-use
  • Performant
  • Simple

Installation

Node.js 8.0.0 or newer is required. Stable: npm i niscord [Not supported] Master: npm i decord/niscord Dev: npm i decord/niscord#dev [Not supported]
